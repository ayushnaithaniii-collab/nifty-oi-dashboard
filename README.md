# Nifty 50 Live OI/PCR Dashboard
Real-time options chain analysis using Upstox API

## What it does
- Fetches live options chain data via Upstox API  
- Calculates Put-Call Ratio (PCR) and Max Pain  
- Visualizes Open Interest (OI) distribution  

## Screenshot
dashboard.png

This dashboard shows real-time Open Interest distribution across strikes.  
PCR helps identify market sentiment, while Max Pain indicates a potential price magnet level.

## Tech Stack
Python, Upstox API, Pandas, Matplotlib, Google Colab

## How to run
1. Clone the repo  
2. Install requirements  
3. Run the script/notebook
